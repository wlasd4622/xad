# XAD广告拦截助手
****
### 一款Chrome插件  [Chrome商店地址](https://chrome.google.com/webstore/detail/xad/aioaekpalodgcjadoicpiepmnjihdbkg?utm_source=chrome-app-launcher-info-dialog) 


#### 功能介绍
    屏蔽网页广告（自动屏蔽百度搜索广告，自定义屏蔽其它网站广告）
    快捷键
    shift+f：激活选择状态
    shift+d：屏蔽选中内容
    shift+w：放大要屏蔽内容的范围
    shift+x：缩小要屏蔽内容的范围
    shift+z：撤销上次的屏蔽操作
    shift+a：撤销此网站的所有屏蔽信息
    shift+c：结束选择状态
